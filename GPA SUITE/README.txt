✅ تشغيل فوري (بدون Node / npm):
1) فك الضغط
2) افتح الملف: index.html (Double click)

✅ التحديثات:
- تصنيف "ممتاز" يبدأ من 3.7 (بدلاً من 3.6).
- صفحة "تحسين التقدير" Workflow جديد:
  * ساعات مُنجزة
  * إجمالي ساعات البرنامج
  * ساعات هذا الفصل
  * المعدل الحالي
  ثم:
  (A) تكتب توقعك هذا الفصل فيحسب المعدل الجديد
  أو
  (B) تكتب هدفك فيحسب المطلوب هذا الفصل.

البيانات تتخزن Offline في LocalStorage.
Backup/Restore داخل Tools → Backup.

=== GitHub Pages (بدون npm) ===
1) افتح GitHub واعمل Repository جديد (Public) مثلاً: delta-gpa-suite
2) ارفع كل ملفات المشروع (فك الضغط ثم Upload)
3) من Settings → Pages:
   - Source: Deploy from a branch
   - Branch: main / (root)
   - Save
4) بعد دقيقة هيظهر لك لينك الموقع: https://YOUR-USER.github.io/REPO/
